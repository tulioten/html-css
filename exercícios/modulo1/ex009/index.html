Trabalhando com Listas

Listas Ordenadas

Usar "ol"(ordered list) para fazer uma lista com ordem sequencial.

  1. Acordar
  2. Tomar Banho
  3. Café da manhã
  4. Escovar os dentes
  5. Atelier
  6. Casa
  7. Almoço

Listas Não Ordenadas.

Usar "ul"(unordered list) para fazer lista sem ordem sequencial, apenas "tópicos"

Comprar:

  • Café
  • Açucar
  • Maça
  • Banana
  • Limão
  • Manteiga
  • Queijo
  • Leite
  • Sal

Lista de Definições

Usado "dl" para iniciar a Lista de Definição

É utilizado "dt" para o (termo) e "dd" para a (explicação), por exemplo:

Termo
Descrição
HTML
Linguagem de marcação para criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação de design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
